# MediQueue - Tutor Booking System

**Live Site:** https://medi-queue-tutor-booking.vercel.app/

A modern full-stack tutor booking platform built with Next.js (Client) and Express.js (Server) with MongoDB. Students can easily find, compare, and book trusted tutors online.

## Key Features

- **Secure Authentication** — Email/Password + Google OAuth login with JWT
- **Tutor Management** — Add, update, and delete tutors with full control
- **Smart Booking System** — Real-time slot management, date restrictions, and auto slot decrement
- **Search & Filter** — Search tutors by name with case-insensitive matching
- **User Dashboard** — View "My Tutors" and "My Bookings" with cancel functionality
- **Modern UI/UX** — Responsive design with Dark/Light theme toggle
- **Separate Client & Server** — Deployed independently on Vercel and Render

## Tech Stack

- **Frontend (Client):** Next.js 16, Tailwind CSS, NextAuth, React Hot Toast
- **Backend (Server):** Express.js, MongoDB, Mongoose, JWT, bcrypt
- **Database:** MongoDB Atlas
- **Deployment:** Vercel (Client) + Render (Server)

## How it Works

1. Users register/login (Email or Google)
2. Browse and search available tutors
3. Book sessions with real-time slot availability
4. Manage personal tutors and bookings from dashboard

Built as a clean, production-ready tutoring marketplace for Bangladesh.
